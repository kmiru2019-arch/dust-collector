# KS 도면표준 — 자동생성 시 준수사항

## 핵심 KS 표준

| 표준 | 내용 | 적용 |
|---|---|---|
| **KS A 0005** | 제도 통칙 | 선·문자·치수의 기본 |
| **KS A 0106** | 도면의 크기 및 양식 | A0~A4 규격, 윤곽선·중심마크 |
| **KS A 0904** | 표제란 | 우측하단 |
| **KS A 0905** | 부품란 | 좌측 또는 우측 |
| **KS A 0906** | 표시방법 | 척도·재질·중량 |

(KS 본문은 유료 표준이므로 직접 호스팅 불가. 본 문서는 일반적 적용 규칙만 기록)

## 도면 규격 (KS A 0106)

| 규격 | 크기 (mm) | 사용 |
|---|---|---|
| A0 | 841 × 1189 | 대형 P&ID, 배치도 |
| A1 | 594 × 841 | P&ID 표준 |
| A2 | 420 × 594 | 단면도 |
| A3 | 297 × 420 | 보고서 첨부 |
| A4 | 210 × 297 | 카탈로그·보고서 |

본 웹앱 자동선택:
- 장비 ≤ 5개 → A4
- 5~15개 → A3
- 15~30개 → A1
- > 30 → A0

## 표제란 (KS A 0904)

우측 하단, 폭 170 mm × 높이 60~80 mm:

```
┌──────────────────────────────────────────────────┐
│ [회사 로고]      회사명        | 도면명           │
│                                ├─────────────────┤
│                                | 부제·시스템      │
├─────────────────┬──────────────┼─────────────────┤
│ 척도   1 : 50   │ 단위    mm   │ 도면번호         │
├─────────────────┼──────────────┤  XYZ-2026-001  │
│ 투상법 [3각법]   │ 일자  YYMMDD │  Rev. A         │
├──────┬──────┬───┴──────┬───────┴─────────────────┤
│ 설계  │ 검토  │ 승인     │  서명·일자              │
│ 김XX │ 이XX  │ 박XX     │                        │
└──────┴──────┴──────────┴────────────────────────┘
```

## 코드 (lib/drawing/dust/title-block.ts)

```typescript
export interface TitleBlockData {
  company: { name: string, logo_url?: string };
  drawing_title: string;
  drawing_subtitle?: string;
  drawing_number: string;          // "DC-2026-001"
  revision: string;                // "A", "B" ...
  scale: string;                   // "1:50", "NTS"
  unit: string;                    // "mm"
  projection: "1st_angle" | "3rd_angle";  // 한국 표준 = 3각법
  date: string;
  designer: string;
  reviewer: string;
  approver: string;
  paper_size: "A0"|"A1"|"A2"|"A3"|"A4";
}

export function generateTitleBlockSVG(data: TitleBlockData): string {
  // SVG 생성 (170mm × 60mm)
  return `<g class="title-block" transform="translate(${pageWidth - 200}, ${pageHeight - 80})">
    <rect width="170" height="60" stroke="black" fill="white"/>
    ...
  </g>`;
}
```

## 선의 종류 (KS A 0005)

| 종류 | 굵기 (mm) | 용도 |
|---|---|---|
| 외형선 | 0.5~0.7 | 보이는 윤곽 |
| 숨은선 | 0.3 | 보이지 않는 윤곽 (점선) |
| 중심선 | 0.2 | 1점쇄선 |
| 치수선·치수보조선 | 0.2 | 가는 실선 |
| 절단선 | 0.5 | 1점쇄선 |
| 가상선 | 0.2 | 2점쇄선 |
| 파단선 | 0.2 | 자유 곡선 |

P&ID 적용:
- 메인 process 라인: 0.7
- Utility: 0.35
- 계장 신호: 0.18 (점선)

## 척도 (Scale)

표준 척도 (KS A 0907):
- 1:1, 1:2, 1:5, 1:10, 1:20, 1:50, 1:100, 1:200, 1:500, 1:1000
- 확대: 2:1, 5:1, 10:1
- NTS (Not To Scale, 척도 없음) — P&ID에 흔함

## 한국어 폰트 (PDF/SVG)

- **본문**: Pretendard (오픈소스, OFL) — 본 웹앱 채택
- **숫자·기호**: Pretendard 또는 Roboto
- **공학기호**: Symbol·STIX 등 폴백

DXF 출력 시:
- TEXT는 단순 ASCII만 — 한글 깨짐
- **MTEXT** 사용 + UTF-8 코드페이지 명시
- AutoCAD에서 SHX 폰트 (예: HangulGothic.shx) 별도 로딩 필요할 수 있음

## 도면 컬러 (선택 — 디지털 표시용)

| 라인 종류 | 컬러 |
|---|---|
| Process gas/dust | 노랑 |
| Compressed air | 청색 |
| Cooling water | 녹색 |
| Slurry/wastewater | 갈색 |
| Steam | 적색 |
| Instrument signal | 검정 (점선) |

흑백 인쇄 시 굵기·종류로 구분.
